# Heart of Life

A single-player heart progression system inspired by Lifesteal for Minecraft.

## Features

1. **Permanent Heart Loss on Death**  
   Every death removes **1 permanent heart**.

2. **Recover Hearts Through Crafting**  
   Regain hearts by crafting **Heart Fragments** into a **Heart of Life**.

   **Heart Fragment Crafting:**
```
D N D
N T N
D N D
```
**Key:**
- **D** = Diamond  
- **N** = Netherite Scrap  
- **T** = Totem of Undying  
- **Stack Size:** 16

   **Heart of Life Crafting:**
```
  F
F H F
  F
```
**Key:**
- **F** = Heart Fragment  
- **H** = Heart of the Sea  
- **Stack Size:** 1

Combine fragments to craft a Heart of Life, which restores a permanent heart when used.

3. **Heart of Life Effects**
- Grants **+1 permanent heart** when used  
- Gives **Regeneration I for 5 seconds**  
- Stacks up to **20** (max health limit)

4. **Heart Limits**
- Maximum: **20 permanent hearts**  
- Minimum: **1 permanent heart** (you never drop below 1)

---

## Installation

1. Download the mod JAR from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/hol) or [Modrinth](https://modrinth.com/mod/hol).  
2. Place the JAR in your `mods` folder for Minecraft with Fabric.  
3. Launch Minecraft.
